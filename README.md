# 🎮 Game Dev Co-Pilot

A multi-agent AI system that turns any game idea 
into playable Python code automatically.

## The Problem

Most people have game ideas but can't code them.
Learning game development takes months. Hiring a 
developer is expensive. No tool exists that goes 
from idea to playable code in minutes.

## The Solution

Three AI agents collaborate in sequence to build 
your game automatically — no coding required.

## How It Works

- **Designer Agent** — turns your idea into a structured game spec
- **Coder Agent** — writes working Python code from the spec
- **Playtester Agent** — reviews the code, finds bugs, scores it

Type "space shooter" → get a playable game in minutes.

## Tech Stack
- Python 3.12
- Google Gemini 2.5 Flash
- Flask web UI

## Run It

git clone https://github.com/Manvi-0912/game-dev-copilot
cd game-dev-copilot
pip install -r requirements.txt
python app.py

Open http://localhost:5000 in your browser.
